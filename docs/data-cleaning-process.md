# Proceso de limpieza y preparación de datos

La base de datos entregada requirió revisión, organización y estructuración previa al análisis. El trabajo incluyó validación de registros, normalización de campos, revisión de categorías y preparación de variables para construir indicadores de gestión y visualizaciones comparables.

## Acciones realizadas
- revisión de estructura general
- limpieza de campos
- normalización de categorías
- validación de consistencia
- preparación de variables para análisis territorial y demográfico
